AsteroidWars
============
Chris Ochynski
Ryan Pawling

To run Asteroid Wars:

1. Unzip this .jar file, navigate to the Asteroids/ directory
2. type make into the command line
3. Navigate to the bin/ directory
4. enter command: 'appletviewer -J-Djava.security.policy=java.policy.applet Asteroids.html'
5. Then play!
